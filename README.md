# polling-app-system
A full-stack online polling application built using React.js, Node.js, Express.js, and MySQL that allows users to create polls, vote securely and view real-time results with a responsive and user-friendly interface.

## Technologies Used
- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MySQL
- Charts: Chart.js

  ## Features
- User registration & login
- Vote on polls (single or multiple choice)
- View poll results with charts
- Admin: create, edit, close, delete polls
- Responsive design for web & mobile

## Project Structure

### Frontend (client)
- Pages: Home, Register, VotePage, ResultPage, MyVotes, AdminDashboard
- Components: Navbar, Footer, PollCard, VoteOption, ChartResult, Notifications

### Backend (server)
- REST APIs: register, login, get polls, vote, get results, admin actions
- Models: Users, Polls, Options, Votes
